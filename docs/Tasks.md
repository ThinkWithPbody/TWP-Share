## Due
```tasks
not done
starts before tomorrow
short mode
sort by priority
sort by status.type
description regex does not match /^$/
NOT (path includes Templater)

group by due
has due date
(scheduled before next week) OR (no scheduled date)
```
## Scheduled
```tasks
not done
starts before tomorrow
short mode
sort by priority
sort by status.type
description regex does not match /^$/
NOT (path includes Templater)

group by scheduled
(scheduled on or before today) OR ((no scheduled date) AND (no due date))
```
